# Student Pass Prediction using Logistic Regression

## Project Overview

This project predicts whether a student will pass or fail using Logistic Regression Classification.

The application is deployed using Streamlit.

---

# Dataset

Dataset used:
- StudentsPerformance Dataset

Features:
- gender
- race/ethnicity
- parental level of education
- lunch
- test preparation course
- reading score
- writing score

Target:
- Passed

Output:
- 1 → Pass
- 0 → Fail
